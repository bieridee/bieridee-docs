BierIdee Eclipse Projekte Readme
=================================

Die beiden Dateien "BierIdee-Back-Eclipse.zip" und "BierIdee-Front-Eclipse.zip"
sind exportierte Eclipse-Projekte, sie können direkt als Projekte importiert
werden.

Zu beachten ist jedoch, dass es sich bei beiden Projekten um Maven-Projekte 
handelt. Das heisst, dass zuvor das Eclipse Maven-Plugin installiert werden
sollte. Um Android-Projekte mit Maven zu benutzen wird noch ein zusätzliches
Plugin benötigt. Diese Plugins können über folgende UpdateSites bezogen 
werden:

    http://download.eclipse.org/technology/m2e/releases

    https://svn.codespot.com/a/eclipselabs.org/m2eclipse-android-integration/updates/m2eclipse-android-integration/

Um das Android-Projekt lauffähig zu haben wird der Android-SDK und das 
ADT-Eclipse-Plugin benötigt.
Diese sind unter folgenden Links zu finden:

Android-SDK:
    http://developer.android.com/sdk/index.html

ADT-Plugin UpdateSite:
    https://dl-ssl.google.com/android/eclipse/


Wenn diese Plugins nicht installiert werden, können die Projekte trotzdem
in Eclipse importiert und betrachtet werden, sie sind jedoch nicht 
lauffähig.
